40. Online Course Progress & Certificate Management System
Detailed Problem Statement

An online learning platform wants to manage learners, courses, modules, quiz scores, progress percentage, payments, and certificates using SQL.

Learners enroll in different online courses, complete modules, attempt quizzes, make course payments, and receive certificates after completing the required progress. The platform wants to track learner performance, course completion, payment status, certificate eligibility, and course popularity.

The system should store:

Learner details
Course details
Enrollment details
Course module details
Module progress details
Quiz score details
Payment details
Certificate details

The system should help the platform answer questions like:

Which learners completed a course?
Which learners are eligible for certificates?
Which course has the most enrollments?
Which learners have pending payments?
What is the average quiz score per course?
Which modules are completed the most?
How much revenue was generated?
Which learners have low progress?
