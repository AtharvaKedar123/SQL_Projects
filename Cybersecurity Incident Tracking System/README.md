# 🔐 Cybersecurity Incident Tracking System

## 📌 Overview
A system to track, manage, and analyze cybersecurity incidents across an organization. It helps monitor threats, assign response teams, and ensure timely resolution.

## 🎯 Objectives
- Track security incidents in real-time
- Assign incidents to analysts
- Monitor resolution status
- Analyze attack patterns

## 🧱 System Components
- Incidents
- Analysts
- Systems
- Threat Types

## 🗂️ Database Design
- incidents
- analysts
- systems
- incident_logs

## ⚙️ Key Features
- Incident severity classification
- Status tracking (Open, In Progress, Resolved)
- Analyst assignment
- Incident history logs

## 📊 Sample Queries
- High severity incidents
- Most targeted systems
- Analyst workload
- Resolution time analysis

## 💡 Use Cases
- SOC monitoring
- Threat response tracking
- Security audits

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
