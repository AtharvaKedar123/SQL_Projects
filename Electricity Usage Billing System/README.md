# 💡 Electricity Usage Billing System

## 📌 Overview
A billing and electricity consumption management system for residential and commercial users.

## 🎯 Objectives
- Track electricity consumption
- Generate monthly bills
- Monitor usage trends
- Detect abnormal usage

## 🧱 System Components
- Consumers
- Meter Readings
- Bills
- Payment Records

## 🗂️ Database Design
- consumers
- meter_readings
- bills
- payments

## ⚙️ Key Features
- Automated bill generation
- Usage tracking
- Payment monitoring
- Consumption analytics

## 📊 Sample Queries
- Highest electricity consumers
- Pending payments
- Monthly billing trends
- Abnormal usage reports

## 💡 Use Cases

- Utility companies
