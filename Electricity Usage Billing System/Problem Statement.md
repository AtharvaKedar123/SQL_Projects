Problem 10: Smart Electricity Usage Billing System
Detailed Problem Statement

An electricity provider wants to manage customer electricity usage and monthly billing digitally using SQL.

Each customer has an electricity meter installed at their house or shop. Every month, the meter reading is recorded. The bill amount is calculated based on units consumed and tariff plan. The system also tracks payment status, due dates, and high-consumption customers.

The system should store:

Customer details
Meter details
Tariff plan details
Monthly meter readings
Electricity bill details
Payment status

The system should help the electricity provider answer questions like:

How many units did each customer consume?
Which customers have pending bills?
Which customers consumed the most electricity?
How much revenue was collected?
Which tariff plan generated the most revenue?
Which bills are overdue?
