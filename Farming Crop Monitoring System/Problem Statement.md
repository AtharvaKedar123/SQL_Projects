Problem 22: Smart Farming Crop Monitoring System
Detailed Problem Statement

A smart farming company wants to manage farms, crops, soil sensors, irrigation schedules, fertilizer usage, and crop health digitally using SQL.

Each farm has different fields where crops are planted. Sensors collect soil moisture, temperature, humidity, and pH values. Based on these readings, irrigation and fertilizer schedules are planned. The system also tracks crop growth stage and health condition.

The system should store:

Farmer details
Farm field details
Crop details
Sensor reading details
Irrigation schedule details
Fertilizer usage details
Crop health records

The system should help the farmer answer questions like:

Which fields need irrigation?
Which crops are unhealthy?
Which fields have low soil moisture?
How much fertilizer was used?
Which crop is grown in each field?
Which fields have abnormal pH levels?
