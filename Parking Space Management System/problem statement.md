Problem 1: Smart Parking Space Management System
Detailed Problem Statement

A smart city parking lot wants to manage its parking slots digitally using SQL.

The parking lot has multiple parking zones such as basement parking, open ground parking, VIP parking, and two-wheeler parking. Each zone contains multiple parking slots. Vehicles enter the parking area, get assigned a slot, stay for some time, and then exit after paying the parking fee.

The system should store:

Parking zone details
Slot details
Vehicle owner details
Parking booking details
Entry and exit time
Parking charges
Slot availability

The system should help the parking owner answer questions like:

Which slots are currently available?
Which vehicles are currently parked?
How much revenue was generated?
Which zone is used the most?
How long did each vehicle stay?
