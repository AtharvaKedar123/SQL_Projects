34. Museum Artifact & Visitor Guide System
Detailed Problem Statement

A museum wants to manage its artifacts, exhibits, visitors, guided tours, ticketing, and artifact maintenance using SQL.

Visitors come to the museum to explore different exhibits, book guided tours, buy tickets, and learn about historical artifacts. The museum management wants to track visitor activity, popular exhibits, ticket revenue, guide assignments, artifact condition, and maintenance history.

The system should store:

Artifact details
Exhibit details
Visitor details
Guide details
Tour booking details
Ticket details
Artifact maintenance details
Exhibit visit records

The system should help the museum answer questions like:

Which exhibit is visited the most?
Which artifacts need maintenance?
Which guide handled the most tours?
How much ticket revenue was generated?
Which visitors booked guided tours?
Which artifact is the oldest?
Which exhibits have the highest number of artifacts?
Which maintenance tasks are pending?
