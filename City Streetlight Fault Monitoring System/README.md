# 🚦 City Streetlight Fault Monitoring System

## 📌 Overview
A system to monitor and manage streetlight faults across a city, ensuring timely maintenance and improved public safety.

## 🎯 Objectives
- Track faulty streetlights
- Assign maintenance teams
- Monitor repair status
- Improve maintenance efficiency

## 🧱 System Components
- Streetlights
- Locations
- Maintenance Teams
- Fault Reports

## 🗂️ Database Design
- streetlights
- locations
- fault_reports
- maintenance_logs

## ⚙️ Key Features
- Fault detection logging
- Maintenance tracking
- Location-based monitoring
- Repair history tracking

## 📊 Sample Queries
- Faults by location
- Pending repairs
- Maintenance team performance
- Frequent fault areas

## 💡 Use Cases
- Smart cities
- Municipal corporations
- Infrastructure management

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
- IoT sensor integration
- Real-time alerts
- Automated fault detection
