Problem 19: Mental Wellness Session Booking System
Detailed Problem Statement

A mental wellness platform wants to manage counselor appointments, therapy sessions, feedback, and payments digitally using SQL.

Users can book sessions with counselors based on specialization such as stress management, anxiety support, career counseling, relationship counseling, or mindfulness coaching. Each session has a scheduled date, time, mode, status, payment record, and feedback score.

The system should store:

User details
Counselor details
Counselor specialization
Session booking details
Session payment details
User feedback details

The system should help the platform answer questions like:

Which sessions are upcoming?
Which counselors have the most bookings?
Which users have pending payments?
Which counselor has the highest feedback rating?
How much revenue was generated?
Which session mode is most preferred?
