Problem 18: Event Ticket Booking & Seat Management System
Detailed Problem Statement

An event management company wants to manage event ticket booking, venue seats, ticket prices, and payments digitally using SQL.

The company organizes different events such as concerts, tech conferences, comedy shows, sports matches, and theatre shows. Each event happens at a venue with multiple seat categories like VIP, Premium, Regular, and Economy. Customers can book seats, make payments, and receive booking confirmation.

The system should store:

Customer details
Venue details
Event details
Seat details
Ticket booking details
Payment details
Seat availability status

The system should help the event company answer questions like:

Which seats are available for an event?
Which customers booked tickets?
Which event generated the most revenue?
Which bookings have pending payments?
Which seat category is booked the most?
How many tickets were sold for each event?
