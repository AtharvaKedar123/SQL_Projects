Problem 11: Smart Hostel Room Allocation System
Detailed Problem Statement

A college hostel wants to manage student room allocation digitally using SQL.

The hostel has multiple blocks, floors, rooms, and beds. Students apply for hostel accommodation based on year, branch, and room preference. The system should allocate rooms, track available beds, manage hostel fees, record complaints, and monitor maintenance requests.

The system should store:

Student details
Hostel block details
Room details
Bed allocation details
Fee payment details
Complaint details
Maintenance status

The system should help the hostel admin answer questions like:

Which students are currently allocated rooms?
Which rooms still have available beds?
Which students have pending hostel fees?
Which block has the most occupied rooms?
Which complaints are unresolved?
Which rooms need maintenance?
