Problem 3: Hospital Emergency Room Management System
Detailed Problem Statement

A hospital wants to manage its emergency room digitally using SQL.

The emergency room receives multiple patients every day. Each patient is assigned a doctor based on the emergency type and severity level. Some patients need beds, some need immediate treatment, and some are discharged after basic treatment. The hospital also needs to track treatment status, bed availability, billing amount, and payment status.

The system should store:

Patient details
Doctor details
Emergency case details
Bed details
Treatment details
Billing details
Payment status

The system should help the hospital answer questions like:

Which patients are currently under treatment?
Which cases are critical?
Which beds are available?
Which doctor handled the most cases?
How much billing revenue was generated?
Which patients have pending payments?
