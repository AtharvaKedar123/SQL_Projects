# 🏥 Hospital Emergency Room Management System

## 📌 Overview
A system for managing emergency room operations, patient queues, and treatment tracking.

## 🎯 Objectives
- Track emergency patients
- Manage doctor allocation
- Reduce waiting times
- Monitor treatment status

## 🧱 System Components
- Patients
- Doctors
- Emergency Cases
- Treatment Logs

## 🗂️ Database Design
- patients
- doctors
- emergency_cases
- treatments

## ⚙️ Key Features
- Emergency case tracking
- Doctor scheduling
- Patient priority management
- Treatment monitoring

## 📊 Sample Queries
- Average waiting time
- Doctor workload analysis
- Critical case reports
- Treatment success trends
