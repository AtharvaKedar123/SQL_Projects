# 🌍 Disaster Relief Resource Distribution System

## 📌 Overview
A system designed to manage and monitor resource distribution across multiple disaster relief camps.

## 🎯 Objectives
- Track camp resources
- Manage supply distribution
- Monitor shortages
- Ensure fair allocation

## 🧱 System Components
- Camps
- Resources
- Volunteers
- Distribution Logs

## 🗂️ Database Design
- camps
- resources
- volunteers
- distribution_records

## ⚙️ Key Features
- Resource allocation tracking
- Camp inventory management
- Volunteer assignment
- Distribution monitoring

## 📊 Sample Queries
- Resource shortages by camp
- Most active camps
- Distribution trends
- Volunteer contribution analysis

## 💡 Use Cases
- Government disaster management
- NGOs
- Emergency response teams

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
- GPS supply tracking
- Real-time monitoring
- Mobile coordination system
