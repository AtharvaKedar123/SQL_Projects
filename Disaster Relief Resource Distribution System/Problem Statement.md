Problem 13: Disaster Relief Resource Distribution System
Detailed Problem Statement

A disaster relief organization wants to manage relief camps, victims, volunteers, supplies, and resource distribution digitally using SQL.

During floods, earthquakes, fires, or other disasters, affected people are moved to relief camps. Each camp needs food, water, medicine, blankets, and volunteers. The organization must track available stock, victim needs, assigned volunteers, and distributed supplies.

The system should store:

Disaster event details
Relief camp details
Victim details
Volunteer details
Supply item details
Resource stock details
Distribution records

The system should help the organization answer questions like:

Which camps need urgent supplies?
How many victims are staying in each camp?
Which supplies are low in stock?
Which volunteers are assigned to each camp?
How many items were distributed?
Which disaster event required the most resources?
