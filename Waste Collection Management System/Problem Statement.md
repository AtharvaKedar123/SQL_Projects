Problem 17: Smart Waste Collection Management System
Detailed Problem Statement

A smart city wants to manage garbage bins, fill-level sensors, pickup routes, drivers, and waste collection records digitally using SQL.

Each garbage bin is installed at a specific location and has a sensor that tracks how full the bin is. When a bin reaches a high fill level, it should be marked for pickup. Drivers are assigned to routes, and collection status is tracked for each bin.

The system should store:

Garbage bin details
Sensor reading details
Driver details
Collection route details
Route-bin mapping details
Waste pickup schedule
Collection status

The system should help the city admin answer questions like:

Which bins are almost full?
Which bins need urgent pickup?
Which driver is assigned to which route?
Which pickups are pending?
Which area generates the most waste?
How many bins were collected successfully?
