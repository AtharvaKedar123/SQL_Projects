Problem 44: Smart Library Seat Booking & Study Room System
Detailed Problem Statement

A smart library wants to manage its seat booking and study room reservation system digitally using SQL.

The library has multiple students, library zones, individual seats, study rooms, booking slots, check-ins, no-shows, and room availability records. Students can book seats or study rooms for specific time slots. The system should track whether a booking is confirmed, checked-in, cancelled, completed, or marked as no-show.

The system should also help library staff monitor seat usage, room occupancy, student booking history, and availability of seats or rooms.

The system should store:

Student details
Library zone details
Seat details
Study room details
Booking slot details
Seat booking details
Room booking details
Check-in details
No-show status
Availability status

The system should help the library answer questions like:

Which seats are currently available?
Which study rooms are booked today?
Which students booked the most slots?
Which bookings became no-shows?
How many seats are available in each zone?
Which rooms are used the most?
Which students checked in successfully?
