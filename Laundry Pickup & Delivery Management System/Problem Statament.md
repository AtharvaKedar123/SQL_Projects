Problem 24: Smart Laundry Pickup & Delivery Management System
Detailed Problem Statement

A laundry service company wants to manage customer laundry orders, pickup requests, washing status, delivery partners, pricing, and payments digitally using SQL.

Customers can request laundry pickup from home. Each order may contain different laundry items such as shirts, jeans, bedsheets, jackets, or curtains. The service tracks pickup time, washing status, ironing status, delivery status, delivery partner, and payment details.

The system should store:

Customer details
Laundry service type details
Laundry order details
Order item details
Pickup and delivery details
Delivery partner details
Payment details

The system should help the laundry company answer questions like:

Which orders are currently in washing?
Which pickups are pending?
Which delivery partner completed the most deliveries?
Which customers have pending payments?
Which service type generated the most revenue?
What is the complete laundry order report?
