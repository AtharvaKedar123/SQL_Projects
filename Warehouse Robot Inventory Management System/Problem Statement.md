Problem 26: Smart Warehouse Robot Inventory Management System
Detailed Problem Statement

A warehouse company wants to manage inventory, storage racks, robot movements, item picking, restocking, and order fulfillment digitally using SQL.

The warehouse uses robots to move products between storage racks and packing stations. Each robot has a battery level, carrying capacity, assigned task, and current status. The system should track products, rack locations, stock levels, robot tasks, customer orders, picked items, and fulfillment status.

The system should store:

Product details
Warehouse rack details
Inventory stock details
Robot details
Robot task details
Customer order details
Order item details
Picking and fulfillment records

The system should help the warehouse answer questions like:

Which products are low in stock?
Which robots are currently available?
Which robot tasks are pending?
Which orders are ready for packing?
Which robot completed the most tasks?
What is the complete inventory movement report?
