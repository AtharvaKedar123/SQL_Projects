# 🚧 Toll Plaza Vehicle & FASTag Payment System

## 📌 Overview
A system to manage toll booth operations, vehicle tracking, and FASTag payment records.

## 🎯 Objectives
- Track toll vehicle entries
- Manage FASTag payments
- Reduce traffic congestion
- Monitor toll revenue

## 🧱 System Components
- Vehicles
- Toll Booths
- FASTag Accounts
- Payment Records

## 🗂️ Database Design
- vehicles
- toll_booths
- fastag_accounts
- payment_records

## ⚙️ Key Features
- FASTag payment tracking
- Toll collection monitoring
- Vehicle entry logging
- Revenue analytics

## 📊 Sample Queries
- Highest toll revenue booths
- Failed FASTag transactions
- Vehicle traffic trends
- Peak traffic hours

## 💡 Use Cases
- Highway toll systems
- Smart transportation systems
- Government road authorities

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
- ANPR vehicle detection
