# 🍔 Online Food Delivery Analytics System

## 📌 Overview
A database system to analyze food delivery orders, restaurant performance, and customer behavior.

## 🎯 Objectives
- Track food orders
- Analyze delivery performance
- Monitor restaurant ratings
- Understand customer trends

## 🧱 System Components
- Customers
- Restaurants
- Orders
- Delivery Agents

## 🗂️ Database Design
- customers
- restaurants
- orders
- delivery_agents

## ⚙️ Key Features
- Order analytics
- Delivery time tracking
- Restaurant performance reports
- Customer behavior analysis

## 📊 Sample Queries
- Most ordered food items
- Top-rated restaurants
- Late delivery reports
- Revenue by restaurant

## 💡 Use Cases
- Food delivery companies
- Restaurant chains
- Delivery analytics teams

## 🛠️ Tech Stack
- SQL
