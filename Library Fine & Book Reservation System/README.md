# 📚 Library Fine & Book Reservation System

## 📌 Overview
A system to manage book reservations, overdue fines, and library member records.

## 🎯 Objectives
- Track borrowed books
- Manage reservations
- Calculate overdue fines
- Monitor library inventory

## 🧱 System Components
- Books
- Members
- Reservations
- Fine Records

## 🗂️ Database Design

- reservations
- fines

## ⚙️ Key Features

- Fine calculation automation
- Member management
- Inventory monitoring

- Overdue book reports
