Problem 20: Apartment Visitor & Security Gate Log System
Detailed Problem Statement

An apartment society wants to manage visitor entry, resident approvals, vehicle entry, security guards, and gate logs digitally using SQL.

Visitors come to meet residents, delivery workers enter for parcel delivery, and vehicles enter/exit through the security gate. The system should track visitor details, resident details, guard details, approval status, entry time, exit time, and vehicle information.

The system should store:

Resident details
Apartment flat details
Visitor details
Security guard details
Visitor entry request details
Vehicle entry details
Gate log details

The system should help the apartment security team answer questions like:

Which visitors are currently inside?
Which visitors are waiting for approval?
Which resident received the most visitors?
Which guard handled the most entries?
Which vehicles entered the apartment?
What is the complete gate activity report?
