Problem 21: Drone Delivery Tracking & Battery Monitoring System
Detailed Problem Statement

A drone delivery company wants to manage drone-based parcel delivery using SQL.

The company uses drones to deliver small packages across different city zones. Each drone has battery capacity, payload limit, current status, and assigned delivery route. The system should track customers, packages, drones, delivery assignments, battery logs, route details, and delivery status.

The system should store:

Customer details
Drone details
Package details
Delivery route details
Drone delivery assignment
Battery monitoring logs
Delivery status

The system should help the company answer questions like:

Which drones are currently available?
Which deliveries are still pending?
Which drone has low battery?
Which drone completed the most deliveries?
Which city zone has the most deliveries?
Which packages failed delivery?
