Problem 30: Smart School Bus Tracking & Student Attendance System
Detailed Problem Statement

A school wants to manage school buses, routes, students, pickup points, driver details, bus attendance, and arrival status digitally using SQL.

Each student is assigned to a bus route and pickup point. The system should track when students board the bus, when they get dropped off, which driver is assigned, whether the bus is delayed, and which students were absent during pickup.

The system should store:

Student details
Bus details
Driver details
Route details
Pickup point details
Student route assignment
Bus attendance records
Bus trip status

The system should help the school answer questions like:

Which students boarded the bus today?
Which students were absent during pickup?
Which buses are currently on route?
Which route has the most students?
Which bus trips were delayed?
What is the complete bus attendance report?
