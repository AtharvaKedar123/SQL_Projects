# ⚡ Electric Vehicle Charging Station Management System

## 📌 Overview
A system to monitor EV charging stations, charging sessions, and energy consumption.

## 🎯 Objectives
- Track charging station usage
- Monitor energy consumption
- Manage charging sessions
- Optimize station performance

## 🧱 System Components
- Charging Stations
- Vehicles
- Charging Sessions
- Payments

## 🗂️ Database Design

- vehicles
- charging_sessions
- payments

## ⚙️ Key Features
- Charging session tracking
- Energy consumption monitoring
- Payment tracking
- Station availability management

## 📊 Sample Queries
