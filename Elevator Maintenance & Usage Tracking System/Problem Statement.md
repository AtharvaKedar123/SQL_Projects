31. Smart Elevator Maintenance & Usage Tracking System
Detailed Problem Statement

A commercial building wants to manage its elevators, floors, daily usage, breakdowns, technicians, maintenance schedules, and repair status using SQL.

Elevators are used by visitors, employees, and staff every day. The building management wants to track elevator trips, identify heavily used elevators, monitor breakdowns, assign technicians, and calculate maintenance costs.

The system should store:

Elevator details
Floor details
Elevator usage logs
Breakdown details
Technician details
Maintenance schedules
Repair status
Maintenance cost details

The system should help the building management answer questions like:

Which elevator is used the most?
Which elevators have the highest breakdown count?
Which technician handled the most repairs?
What is the total maintenance cost?
Which elevators are currently under repair?
Which floors receive the most elevator traffic?
Which elevators need upcoming maintenance?
What is the average repair cost per elevator?
