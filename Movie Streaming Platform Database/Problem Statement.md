Detailed Problem Statement

A movie streaming platform wants to manage its users, movies, subscriptions, watch history, and ratings using SQL.

Users can subscribe to different plans, watch movies from different genres, and rate or review movies after watching them. The platform wants to analyze user activity, popular movies, top-rated content, subscription revenue, and watch time.

The system should store:

User details
Subscription plan details
User subscription details
Movie details
Watch history
Movie ratings and reviews
Payment status

The system should help the platform answer questions like:

Which movies are watched the most?
Which users watched the most content?
Which movies have the highest rating?
How much subscription revenue was generated?
Which users have active subscriptions?
Which genre is most popular?
