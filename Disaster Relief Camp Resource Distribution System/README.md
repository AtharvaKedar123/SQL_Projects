# 🌍 Disaster Relief Camp Resource Distribution System

## 📌 Overview
A system to manage resource distribution across multiple disaster relief camps, ensuring efficient allocation and tracking.

## 🎯 Objectives
- Track relief camp resources
- Manage supply distribution
- Monitor shortages
- Ensure fair allocation

## 🧱 System Components
- Camps
- Resources
- Volunteers
- Distribution Logs

## 🗂️ Database Design
- camps
- resources
- volunteers
- distribution_records

## ⚙️ Key Features
- Resource allocation tracking
- Camp-wise inventory management
- Volunteer assignment
- Distribution monitoring

## 📊 Sample Queries
- Resource shortages by camp
- Most active camps
- Distribution trends
- Volunteer contribution

## 💡 Use Cases
- Government disaster management
- NGOs
- Emergency relief operations

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
- GPS tracking of supplies
- Real-time updates
- Mobile coordination system
