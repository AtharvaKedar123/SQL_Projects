# ✈️ Airport Baggage Tracking & Lost Luggage System

## 📌 Overview
A system to track passenger baggage across airports, manage lost luggage, and improve baggage handling efficiency.

## 🎯 Objectives
- Track baggage movement
- Handle lost luggage cases
- Improve baggage tracking accuracy
- Reduce delays and misplacement

## 🧱 System Components
- Passengers
- Baggage
- Flights
- Tracking Logs

## 🗂️ Database Design
- passengers
- baggage
- flights
- baggage_tracking
- lost_reports

## ⚙️ Key Features
- Baggage tracking across checkpoints
- Lost luggage reporting
- Status updates
- Flight-wise baggage mapping

## 📊 Sample Queries
- Lost baggage cases
- Delayed baggage
- Passenger baggage history
- Flight-wise baggage count

## 💡 Use Cases
- Airports
- Airlines
- Logistics tracking

## 🛠️ Tech Stack
- SQL

## 🚀 Future Enhancements
- RFID tracking integration
- Real-time passenger notifications
- Mobile app support
