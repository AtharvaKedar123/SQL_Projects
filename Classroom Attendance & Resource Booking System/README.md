# 🎓 Classroom Attendance & Resource Booking System

## 📌 Overview
A system to manage classroom attendance and allow booking of shared resources like labs and equipment.

## 🎯 Objectives
- Track student attendance
- Manage classroom resources
- Enable booking system
- Avoid scheduling conflicts

- Resources
- Bookings
