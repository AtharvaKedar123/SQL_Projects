35. Smart Classroom Attendance & Resource Booking System
Detailed Problem Statement

A college wants to manage smart classrooms, students, faculty, classes, attendance, classroom resources, lab bookings, and faculty assignments using SQL.

Students attend different classes in different smart classrooms. Faculty members are assigned to courses, and classrooms contain resources such as projectors, smart boards, computers, and lab equipment. The college wants to track attendance, classroom usage, resource bookings, lab availability, and faculty workload.

The system should store:

Student details
Faculty details
Course details
Classroom details
Class schedule details
Attendance records
Classroom resource details
Resource booking details
Lab booking details
Faculty assignments

The system should help the college answer questions like:

Which students have low attendance?
Which classrooms are used the most?
Which faculty teaches the most classes?
Which resources are booked most often?
Which lab bookings are pending?
What is the attendance percentage of each student?
Which courses have the highest attendance?
Which classrooms have smart resources available?
